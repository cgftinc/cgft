
## Creating a RAG Environment

An environment defines tools and rewards for your RAG task.

### 1. Define Tools

```python
from benchmax.envs.base_env import BaseEnv, ToolDefinition

class SearchEnv(BaseEnv):
    def __init__(self, api_key, corpus_id, base_url, dataset_path):
        self._api_key = api_key
        self._corpus_id = corpus_id
        self._base_url = base_url
        self._dataset_path = dataset_path

        # Define search tool
        search_tool = ToolDefinition(
            name="search_corpus",
            description="Search corpus using BM25",
            input_schema={
                "type": "object",
                "properties": {
                    "query": {"type": "string"},
                    "metadata": {"type": "object"},
                    "limit": {"type": "integer"},
                },
                "required": ["query"],
            },
        )
```

### 2. Implement Tool Execution

```python
async def _search_corpus_tool(self, query: str,
                               metadata: dict = None,
                               limit: int = 10) -> str:
    """Execute BM25 search and return formatted results"""
    url = f"{self._base_url}/api/corpora/{self._corpus_id}/search"

    async with aiohttp.ClientSession() as session:
        resp = await session.post(url, json={
            "query": query,
            "metadata": metadata,
            "limit": limit
        })
        results = await resp.json()

    # Format results for model
    return format_search_results(results)
```

### 3. Define Reward Function

```python
async def compute_reward(self, rollout_id: str,
                        completion: str,
                        ground_truth: Any,
                        **kwargs) -> dict[str, float]:
    """Reward based on chunk overlap"""
    reference_chunks = kwargs.get("reference_chunks", [])
    retrieved_text = extract_retrieved_chunks(completion)

    # Compute % of reference chunks found
    overlap_score = chunk_overlap(retrieved_text, reference_chunks)

    return {"chunk_overlap": overlap_score}
```

### 4. Bundle and Upload

```python
from benchmax.bundle.bundler import bundle_env

# Bundle environment with dependencies
payload = bundle_env(
    SearchEnv,
    pip_dependencies=["aiohttp"],
    extra_metadata={"description": "BM25 search environment"}
)

# Upload to platform
storage_client.upload_file(
    path=f"envs/search/{hash}/search-env.bmxp",
    content=payload.to_bytes()
)
```

## Creating Training Data

Training data consists of **question-answer-reference** tuples.

### Data Format

```json
{
  "question": "how to setup reverse proxy for posthog",
  "answer": "Configure nginx with proxy_pass to...",
  "reference_chunks": [
    {"chunk_id": "...", "content": "nginx configuration..."}
  ],
  "qa_type": "single_hop"
}
```

### 1. Chunk Your Corpus

```python
from synthetic_data_prep.chunkers.markdown import MarkdownChunker

chunker = MarkdownChunker(
    min_char=1024,
    max_char=2048,
    chunk_overlap=128
)

collection = chunker.chunk_folder("./docs", file_extensions=[".md", ".mdx"])
```

### 2. Upload to Corpus

```python
from synthetic_data_prep.corpus.client import CorpusClient

corpus_client = CorpusClient(api_key=API_KEY, base_url=BASE_URL)
corpus = corpus_client.get_or_create_corpus("my-docs")

corpus_client.upload_chunks(
    corpus_id=corpus.id,
    collection=collection,
    batch_size=100
)
```

### 3. Generate Q&A Pairs

**Single-hop** (one chunk contains answer):

```python
from synthetic_data_prep.qa_generation.helpers import generate_single_hop_batch

single_hop_dataset = generate_single_hop_batch(
    collection=collection,
    client=openai_client,
    model="grok-4-fast",
    num_samples=40
)
```

**Multi-hop** (multiple chunks needed):

```python
multi_hop_dataset = generate_multi_hop_batch(
    collection=collection,
    client=openai_client,
    corpus_client=corpus_client,
    corpus=corpus,
    num_samples=5
)
```

### 4. Save and Upload

```python
from synthetic_data_prep.qa_generation.storage import save_qa_dataset_jsonl

# Combine datasets
combined = single_hop_dataset.merge(multi_hop_dataset)

# Save as JSONL
save_qa_dataset_jsonl(combined, "outputs/qa_dataset.jsonl")

# Upload to platform
storage_client.upload_local_file(
    path="datasets/search/{corpus_id}/qa-dataset.jsonl",
    file_path="outputs/qa_dataset.jsonl"
)
```

## Running Experiment

Launch a training job with your environment and dataset.

### 1. Configure Training

```python
from synthetic_data_prep.trainer import TrainerClient

trainer_client = TrainerClient(api_key=API_KEY, base_url=BASE_URL)

job = trainer_client.launch_job(
    job_type="search",
    args={
        "dataset": "datasets/search/{corpus_id}/qa-dataset.jsonl",
        "env_path": "envs/search/{hash}/search-env.bmxp",
        "env_args_path": "envs/search/{hash}/search-env-kwargs.json"
    }
)
```

### 2. Monitor Training

```python
# Check job status
status = trainer_client.get_job_status(job.id)

# View metrics
metrics = trainer_client.get_job_metrics(job.id)
```

### 3. Training Hyperparameters

Common settings (configured server-side):

- **Base model**: Claude 3.5 Haiku or similar
- **Learning rate**: 1e-5 to 5e-5
- **Batch size**: 8-32 examples
- **Training steps**: 500-2000 depending on dataset size
- **Reward scaling**: Normalize rewards to [-1, 1]

## Running Baseline Eval

Evaluate your trained model against the validation set.

### Validation Split

```python
class SearchEnv(BaseEnv):
    def get_train_val_split(self, train_ratio=0.7, seed=42):
        """Load dataset and return stratified splits"""
        dataset = load_dataset("json", data_files=self._dataset_path)

        # Split by qa_type to ensure balanced evaluation
        train_dataset, val_dataset = stratified_split(
            dataset,
            stratify_by="qa_type",
            train_ratio=train_ratio
        )

        return train_dataset, val_dataset
```

### Evaluation Metrics

**Retrieval metrics**:
- Chunk overlap (% of reference chunks retrieved)
- Precision@K (% of retrieved chunks that are relevant)
- Mean reciprocal rank (position of first relevant chunk)

**Answer metrics**:
- Exact match (parsed answer == ground truth)
- F1 score (token overlap)
- LLM judge score (semantic correctness)

### Run Evaluation

```python
# Automatic evaluation during training
eval_results = trainer_client.get_eval_results(job.id)

# Manual evaluation
from benchmax.eval import evaluate_model

results = await evaluate_model(
    model=trained_model,
    env=search_env,
    dataset=val_dataset,
    metrics=["chunk_overlap", "answer_f1"]
)
```

### Compare to Baseline

Track improvement over base model:

```python
baseline_results = evaluate_model(base_model, search_env, val_dataset)
finetuned_results = evaluate_model(trained_model, search_env, val_dataset)

improvement = {
    metric: finetuned_results[metric] - baseline_results[metric]
    for metric in finetuned_results
}
```
