# sharing and serving your model (experimental!)
after training your model, you'll want to deploy it for use or share it with your team.
you can do it directly via the experiment platform ui or through the openai client.

## platform ui
- head to your experiment on [cgft](app.cgft.io)
- go to the playground tab

## openai client
- copy the id from the right most button on the experiment site
- call the openai client with `base_url=app.cgft.io/inference` and model_id set to experiment id you copied.

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://app.cgft.io/inference",
    api_key="your-api-key"
)

# use your fine-tuned model
response = client.chat.completions.create(
    model="your-experiment-id-here",  # paste the experiment id you copied
    messages=[
        {"role": "user", "content": "your prompt here"}
    ]
)
print(response.choices[0].message.content)
```
